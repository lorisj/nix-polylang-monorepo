# Generated code from protobufs
Important: Do not edit code in `./generated`. These are generated code stubs using `buf`.

## Generating code
In `packages/proto`, run `buf generate` to generate code stubs for all languages.


## Dependencies
These are from [here](https://buf.build/plugins/).

### Typescript
If you use `bufbuild/es`: `@bufbuild/protobuf@^2.8.0`

If you use `grpc/web`: `grpc-web@2.0.1` and `google-protobuf@^3.21.4`. You also need to set up a proxy. See [here](https://grpc.io/docs/platforms/web/basics/).


### Python

If you use `grpc/python`: `pip install grpcio`

If you use `mypy-grpc`: see [here](https://buf.build/community/nipunn1313-mypy-grpc?version=v3.6.0)

If you use `betterproto`: `pip install betterproto`
